# Jaqaru Digital Corpus

A repository dedicated to compiling Jaqaru texts and building a Spanish-Jaqaru parallel corpus.

## Acknowledgments

Special thanks to linguist **Neli Belleza Castro** for her valuable contributions to this project, sharing her knowledge, guidance and the Jaqaru vocabulary.

## Requirements

- Python 3.10+

### Virtual environment

Create a virtual environment and install dependencies.

```
python -m venv <venv_name>
```
